使用說明

如何取得你的 Threads 資料？

  1. 前往 Instagram 設定 → 帳號中心
  2. 選擇「你的資訊和權限」→「下載你的資訊」
  3. 選擇 Threads 資料，格式選 JSON
  4. 下載後直接上傳 ZIP 檔案
